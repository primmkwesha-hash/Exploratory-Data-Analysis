# Patient Readmission Exploratory Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a patient readmission dataset using Python in Google Colab.

## Objectives
- Explore and understand the dataset
- Identify patterns in patient readmissions
- Create meaningful visualisations
- Communicate findings through observations

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Data inspection
- Age group analysis
- Gender analysis
- Readmission analysis
- Length of stay analysis
- Correlation analysis
- Data visualisation

## Key Findings
- Exploratory analysis showed that non-readmitted patients were more numerous than readmitted patients across all age groups.
- Adults represented the largest age group, while Youth represented the smallest.
- No clear relationship was observed between age and length of stay, and length of stay was similar between readmitted and non-readmitted patients.
- Blood pressure distributions were similar across gender and readmission status, suggesting that blood pressure alone may not strongly distinguish readmitted       patients.
- Blood sugar levels varied across age groups without a clear age-related pattern.
- Previous admission patterns differed across age groups, with Adults and Seniors generally having higher patient counts.



## Conclusion
Overall, the analysis found that patient readmission was not strongly associated with a single demographic or clinical variable examined. Seniors formed the largest age group, while non-readmitted patients generally outnumbered readmitted patients across age groups. Blood pressure distributions were similar across genders and readmission statuses, while blood sugar levels showed some variation between age groups. The analysis demonstrates the importance of using multiple variables to understand patient patterns and identify potential factors associated with readmission.
